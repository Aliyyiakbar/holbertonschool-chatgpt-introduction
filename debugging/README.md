# Debugging

This directory contains my solutions for the debugging tasks in the
**"ChatGPT - Introduction"** project.

The goal of this part of the project is to use ChatGPT to identify bugs,
understand why code fails, and apply correct fixes in Python and HTML/JavaScript
examples.

## Files

- `factorial.py`  
  Fixes the factorial program so it correctly computes and prints the factorial
  of the command-line argument.

- `print_arguments.py`  
  Prints all command-line arguments passed to the script.

- `change_background.html`  
  Implements a simple interface for changing the page background color.

## Usage

### Python examples

```bash
chmod +x factorial.py
./factorial.py 5
```

```bash
chmod +x print_arguments.py
./print_arguments.py Hello Holberton
```

### HTML example

Open `change_background.html` in a browser and interact with the page.
