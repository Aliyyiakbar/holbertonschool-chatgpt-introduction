# ChatGPT - Introduction

This repository contains my solutions for the Holberton School project
**"ChatGPT - Introduction"**.

The goal of this project is to explore how ChatGPT can be used to improve
software development workflows through debugging, documentation, error handling,
and simple task automation across different technologies.

## Objectives

- Practice using ChatGPT to identify and fix bugs in existing code
- Improve code quality through clearer logic and better structure
- Use AI assistance for documentation and repetitive development tasks
- Build confidence in testing and verifying AI-generated solutions

## Repository Structure

- `debugging/`  
  Contains the practical debugging exercises completed for this project.

## Technologies Used

- Python 3
- HTML
- JavaScript
- ChatGPT for debugging and automation support

## Notes

This project focuses on using AI as a coding assistant, while still testing,
verifying, and critically reviewing every generated or suggested solution.
